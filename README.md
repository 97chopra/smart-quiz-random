## Random Smart Quiz Game

A fun quiz game I built on my own using HTML, CSS and JavaScript.

## What It Does

- Shows random questions covering Maths, GK and Science
- 4 multiple choice answers per question
- 3 lives — lose one for every wrong answer
- Hint button if you get stuck
- Quit anytime and come back to continue from where you left off

##  Built With

- HTML
- CSS
- JavaScript
