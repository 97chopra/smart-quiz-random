# 🧠 Smart Quiz Game

A fun and interactive quiz game built with vanilla JavaScript. Test your knowledge across Maths, General Knowledge and Science — with lives, hints, and the ability to pick up where you left off!

## ✨ Features

- ❓ **Random Questions** — Questions covering Maths, General Knowledge and Science
- 🎯 **Multiple Choice** — 4 answer options per question
- ❤️ **3 Lives System** — Lose a life for every wrong answer
- 💡 **Hint Button** — Get a hint when you're stuck
- 💾 **Resume Progress** — Quit anytime and continue from where you left off
- 📱 **Responsive Design** — Works on mobile and desktop


2. Open `index.html` in your browser — no installs needed! ✅

---

## 🎮 How to Play

1. Click **Start Quiz** to begin
2. Read each question and select your answer from 4 choices
3. Use the **Hint** button if you get stuck (use wisely!)
4. You have **3 lives** — each wrong answer costs one
5. Click **Quit** to save your progress and come back later
6. Try to complete all questions without losing all your lives!

## 💡 Key Concepts Practised

**JavaScript**
- Randomising question order using array shuffle
- Tracking game state (score, lives, current question)
- Saving and loading progress using `localStorage`
- DOM manipulation for dynamic question and answer rendering
- Event listeners for interactive button clicks

**CSS**
- Responsive layout for all screen sizes
- Smooth transitions and hover effects
- Visual feedback for correct and incorrect answers
